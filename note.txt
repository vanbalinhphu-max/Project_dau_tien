Repository (Repo - kho chứa)

- Local Repository: nằm trực tiếp trên máy, sẽ nằm trong folder .git
- Remote Repository: kho lưu trữ trên 1 máy chru khác (Github, Gitlab)




Commit
- là 1 bản sao lưu (snapshot) trạng thái của tất cả các file trong dự án tại MỘT THỜI ĐIỂM CỤ THỂ



Khu vực làm việc (Working Area)


- working directory (thư mục làm việc)
- staging area (khu vực chuẩn bị / index)
- repository: (.git repository)


Cách commit đúng cách:
ví dụ:

TỐt
Fix: Tính đúng logic cho tổng tiền thanh toán
Feat: thêm chức năng reset password


Không tốt
update files
fix bugs and add new features
